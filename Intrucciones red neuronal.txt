Intrucciones para redes neuronales con Keras
---------------------------

1-Actualizar pip3
pip3 install -U ip

2- Instalar scipy(numpy etc.)
python -m pip install --user numpy scipy matplotlib ipython jupyter pandas sympy nose
Tuto:https://www.aprenderpython.net/scipy-funciones-principales/

	-- instalar tk inter si es necesario
	sudo apt-get install python3-tk
3- Instlar tensorflow
sudo apt-get install python3-pip python3-dev 
pip3 install tensorflow	
Tuto:https://www.tensorflow.org/install/install_linux#InstallingNativePip

4- installar Keras
pip3 install keras
